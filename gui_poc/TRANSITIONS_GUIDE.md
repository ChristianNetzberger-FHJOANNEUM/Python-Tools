## ⚙️ **Settings:**

### **Effect (Dropdown):**
```
Choose transition type:

```

### **Photo Duration (Speed):**
```
How long each photo shows:
- 2s (quick review)
- 3s (normal)

- 7s (slow, immersive)
```

### **Transition Speed:**
```
How fast the transition happens:
- 0.5s (snappy)
- 1s (default, smooth)
- 1.5s (slower)
- 2s (very smooth)
- 3s (dramatic)

Note: Does NOT affect Ken Burns!
```

### **Loop:**
```
ON: Restart from beginning
OFF: Stop at last photo
```

## 🎬 **Ken Burns Effect - Special:**

### **Was ist das?**
```
Named after documentary filmmaker
Slow zoom + pan on static photos
Creates motion & life
Like in PBS documentaries!
```

### **Wie funktioniert's?**
```
1. Photo appears (fade in)
2. During display: Slowly zooms & pans
3. Scale: 1.0 → 1.15 (15% zoom)
4. Pan: Random direction (-8% to +8%)
5. Duration: Full photo duration (e.g., 5s)
6. Smooth ease-out animation
```

### **Best Settings:**
```
Photo Duration: 7-10s (needs time!)
Transition Speed: 1-2s
Loop: ON (for continuous display)
Photos: Landscapes, details, people
```

### **Why it's awesome:**
```
✅ Makes static photos dynamic
✅ Professional documentary look
✅ Viewer stays engaged
✅ Reveals photo details
✅ Cinematic storytelling
```

## 🔧 **Technical Details:**

### **CSS Variables:**
```css
--transition-speed: 1s       /* Transition duration */
--photo-duration: 5s         /* Photo display time */
--ken-x: -5%                 /* Ken Burns pan X */
--ken-y: -5%                 /* Ken Burns pan Y */
```

### **Animation Types:**
```css
/* Fast transitions (Slide, Zoom, Rotate) */
animation: effectName 1s ease-out;

/* Ken Burns (during display) */
animation: kenBurns 5s ease-out;
```

### **Random Algorithm:**
```javascript
// Consistent random per photo index
const randomIndex = (index * 7) % effects.length;
transition = effects[randomIndex];

// Ken Burns pan directions
const panX = directions[(index * 3) % directions.length];
const panY = directions[(index * 5) % directions.length];
```

## 🎓 **Tips & Tricks:**

### **Transition Speed vs. Photo Duration:**
```
Transition Speed: How fast CHANGE happens (0.5-3s)
Photo Duration: How long photo DISPLAYS (2-10s)

Fast transition + Long duration = Smooth, immersive
Slow transition + Short duration = May feel sluggish
```

### **Ken Burns Best Practices:**
```
✅ Use 7-10s duration (needs time!)
✅ Works best on high-res photos
✅ Great for landscapes & details
✅ Can be combined with fade-in
✅ Random pan keeps it interesting
```

### **Random Mode Strategy:**
